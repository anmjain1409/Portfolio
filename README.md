🚀 Introduction

This is my personal portfolio website, built to showcase my background as a Computer Science Engineer (AIML/DL specialization) and a PG-DAC trainee at CDAC Hyderabad.

The portfolio highlights:

My technical skills

Internship experience (ImpactQA, Ciena, Ypsilon)

Projects and hands-on work

A functional contact form using EmailJS

The website is fully responsive, dark-mode enabled, and optimized for recruiters and hiring managers.

⚙️ Tech Stack

React.js – Component-based UI

Vite – Fast build & dev environment

Tailwind CSS – Utility-first styling

Lucide Icons – Modern icon library

Radix UI – Accessible UI primitives

EmailJS – Contact form email integration

Git & GitHub – Version control

Vercel – Deployment

⚡ Quick Start
Prerequisites

Node.js (v18+ recommended)

Git

Clone the Repository
git clone https://github.com/anmjain1409/react-portfolio.git
cd react-portfolio

Install Dependencies
npm install

Run Locally
npm run dev


Open 👉 http://localhost:5173

☁️ Deployment
Deploy on Vercel

Push code to GitHub

Go to https://vercel.com

Import your repository

Click Deploy

Your portfolio will be live at:

https://portfolio-g45h.vercel.app/
